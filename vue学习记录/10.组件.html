一个组件的 data 选项必须是一个函数，因此每个实例可以维护一份被返回对象的独立的拷贝。

组件必须先注册以便 Vue 能够识别

注册类型：全局注册和局部注册

    全局注册:取名字，接收传值，写模板 。
  • Vue.component('my-component-name', {
    // ... options ...
    })
    局部注册：写成一个对象，通过components关键字在根组件中注册。

  • var cpName = {
    props:['item'],//组件的传值
    template:`//template中只能有一个根元素 <li v-for='item of list'>{{item}}</li>`
    }

  • 局部组件要在根组件中注册：
    components:{
    cpName
    }

例如: Vue.component('cpName',{
props:['item'],//组件的传值
template:` <li v-for='item of list'>{{item}}</li>
}) `

使用组件（模板） ：<cp-name></cp-name>
